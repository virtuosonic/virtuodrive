virtuodrive is a distortion effect for electric guitar, available in jesusonic and vst format.

It can be used inside hosts like Cubase, REAPER, Nuendo, etc...

Download

You can download virtuodrive from the file manager at sourceforge. There you should find a zip file containing virtuodrive.dll

To install uncompres virtuodrive.dll to your DAW's vst folder, if you can't open the zip file i sugest you using 7-Zip
